Double pendulum simulation



Source Code

Description:

In this simulation I use knowledge from my coding modules during my University Course to aid me in making a chaos diagram of a double pendulum. It tracks the movement to show the beauty of chaos using lagrangian mechanics. It functions using non-linear dynamics meaning the pendulums angle and derivatives are not proportinal. The butterfly effect is observed as the tiniest difference in initial condition can lead to incredibly differnet trajectories as time passes.

Skills used:

Simulation Coding and Lagrangian Mechanics
